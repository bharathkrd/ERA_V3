# MNIST Model: The "Just Right" Porridge 🥣

## What's Cooking? 🤔

We're building a neural network for MNIST that's like Goldilocks' porridge - not too big, not too small, but just right! 
Our mission (should you choose to accept it) is to create a model that:

- Has between 15,000 and 25,000 parameters (because we're picky like that)
- Achieves 95%+ accuracy in just ONE epoch (we're speed demons! 🏎️)

## The Secret Sauce 🥫

Our model is like a sandwich:
- Two Conv2D layers (the bread 🍞)
- Some MaxPooling2D (the lettuce 🥬)
- Dense layers for the finale (the cheese 🧀)

All carefully crafted to hit that sweet spot of parameters and accuracy!

## Testing, Testing, 1-2-3 🎤

We've got GitHub Actions set up to be our strict but fair judge. It checks:
1. If our model has the right number of parameters (15k-25k)
2. If it actually achieves that sweet 95%+ accuracy in one epoch
